━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Layout
  • 2×6 day grid, each day has header / body (2-col icon grid) / footer
  • Meetings block (top) + divider line + tasks block (bottom) within each day
  • "Dog day" 🐕 banner on personal days (orange strip above header)

Item tiles
  • Color tile with centered emoji icon
  • Hover/tap slides in the label text (max-width transition)
  • Meetings get a 3px left border; tasks don't
  • Upcoming pulse animation for imminent items

Top bar
  • "📅 My Week" title
  • Week nav (‹ Week 9 · Feb 23 – Mar 1, 2026 ›)
  • 🔗 Share button + v0.3 version badge + ⚙️ settings icon

Legend bar (below top bar)
  • Color → customer name mapping
  • Icon key: 🤝 🏢 🖥️ 💬 = meeting / ✉️ 📋 💻 📝 = task
  • "🔒 N hidden" badge — click to open hidden items panel
  • "Tap · hold to edit" hint

Hidden / private items
  • Hidden items are COMPLETELY INVISIBLE in the main grid (display:none)
  • The 🔒 N hidden badge in the legend shows the count
  • Clicking the badge opens the HIDDEN ITEMS PANEL (bottom sheet):
      - Lists all hidden items: icon + title + Unhide button
      - Clicking Unhide makes the item visible again in the grid
      - Badge disappears when count reaches 0

Version tracking
  • APP_VERSION constant in JS (e.g. 'v0.3.0') + APP_VERSION_DATE
  • Subtle version badge in the top bar (v0.3)
  • Clicking it opens the VERSION HISTORY PANEL (bottom sheet):
      - Lists all versions with tag / date / description
  • Always update version + date when making significant changes

Tooltip (tap/click)
  • Floating dark tooltip: label + ✏️ Edit button
  • Auto-hides after 2.6 seconds

Edit bottom sheet (tap Edit or long-press 480ms on touch)
  • Title input
  • Customer selector (color chips)
  • Type: Meeting / Task (segmented)
  • Icon picker (16 emoji grid)
  • Private toggle (hides item completely — managed via legend badge)
  • Google Calendar note:
      "🔁 Recurring events — managed via Google Calendar, syncs here automatically"
      [Open →] link to calendar.google.com
  • NO in-app recurring settings (frequency, etc.) — all done in Google Calendar
